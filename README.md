# Python Console Quiz game

## About

This is a simple quiz game made using Python. The user enters a number to choose a question, types the answer, and gets instant feedback. The quiz can be ended anytime by entering **0**.

## Features

- 9 general knowledge questions
- Checks if the answer is correct
- Shows the correct answer if the user is wrong
- Easy to use

## Technologies Used

- Python

## Concepts Used

- Dictionary
- While loop
- If-else statements
- User input

## How to Run

1. Download or clone this project.
2. Open the project folder.
3. Run the program:

```bash
python quiz_game.py
```
